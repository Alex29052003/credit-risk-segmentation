# Анализ кредитного риска и предсказание дефолта клиентов

## Описание проекта
Цель проекта — предсказать вероятность дефолта клиента банка, чтобы заранее выявлять рискованных клиентов и снижать финансовые потери.

## Данные
Использованы данные клиентов банка, включающие:
— возраст  
— кредитный лимит  
— история платежей  
— суммы платежей  
— статус дефолта  

Каждая строка — информация по клиенту.

## Анализ данных (EDA)
В ходе анализа было выявлено:
— клиенты с просрочками (PAY_0) чаще допускают дефолт  
— меньшие платежи связаны с более высоким риском  
— возраст и кредитный лимит также влияют на поведение клиентов  

## SQL-анализ
Данные были загружены в SQLite, после чего выполнялись SQL-запросы для:
— группировки клиентов  
— анализа распределений  
— проверки гипотез  

## Моделирование
Были обучены модели:
— Logistic Regression  
— Random Forest  

Данные были разделены на обучающую и тестовую выборки (80/20).

## Оценка моделей
Модели оценивались с помощью accuracy и classification report.

Было выявлено, что accuracy может быть обманчивой метрикой при дисбалансе классов.

## Работа с порогом (ключевой момент)
Для улучшения выявления дефолтных клиентов:
— использовались вероятности модели  
— порог был снижен с 0.5 до 0.3  

Это позволило увеличить recall и находить больше рискованных клиентов.

## Результаты
— Random Forest показал более стабильный результат  
— увеличение recall позволило лучше выявлять дефолтных клиентов  

## Бизнес-выводы
Результаты проекта можно использовать для:
— раннего выявления клиентов с высоким риском дефолта  
— корректировки кредитной политики  
— снижения финансовых потерь
