# Анализ кредитного риска и предсказание дефолта клиентов

## Описание проекта
Цель проекта — предсказать вероятность дефолта клиента банка, чтобы заранее выявлять рискованных клиентов и снижать финансовые потери.В рамках проекта реализован полный цикл: анализ данных, обучение моделей и создание API-сервиса для использования модели.
## Данные
Использованы данные клиентов банка, включающие:
— возраст  
— кредитный лимит  
— история платежей  
— суммы платежей  
— статус дефолта  
Каждая строка — информация по клиенту.
## Анализ данных (EDA)
В ходе анализа было выявлено:
— клиенты с просрочками (PAY_0) чаще допускают дефолт  
— меньшие платежи связаны с более высоким риском  
— возраст и кредитный лимит также влияют на поведение клиентов  
## SQL-анализ
Данные были загружены в SQLite, после чего выполнялись SQL-запросы для:
— группировки клиентов  
— анализа распределений  
— проверки гипотез  
## Моделирование
Были обучены модели:
— Logistic Regression  
— Random Forest  
Данные были разделены на обучающую и тестовую выборки (80/20).
## Оценка моделей
Модели оценивались с помощью accuracy и classification report.
Было выявлено, что accuracy может быть обманчивой метрикой при дисбалансе классов.
## Работа с порогом (ключевой момент)
Для улучшения выявления дефолтных клиентов:
— использовались вероятности модели  
— порог был снижен с 0.5 до 0.3  
Это позволило увеличить recall и находить больше рискованных клиентов.
## Разработка сервиса
Дополнительно был реализован API-сервис на FastAPI для использования модели.
Сервис принимает данные клиента и возвращает:
— вероятность дефолта  
— бинарный прогноз  
— уровень риска  
Пример ответа:
json
{
  "default_probability": 0.46,
  "prediction": 0,
  "risk_label": "low_risk"
}
Сервис позволяет использовать модель в реальных задачах, например для оценки новых клиентов при принятии решения по кредиту.
## Запуск проекта
1) Установить зависимости:
pip install -r requirements.txt  
2) Обучить модель:
python src/train_model.py  
3) Запустить API:
uvicorn src.app:app --reload  
4) Открыть в браузере:
http://127.0.0.1:8000/docs
